# Ejercicio 1 – Iniciando tu primer repositorio
## Objetivo
Inicializar y versionar un proyecto básico.

## Instrucciones
1. Crea una carpeta:
   ```bash
   mkdir mi_repositorio && cd mi_repositorio
   ```
2. Inicializa Git:
   ```bash
   git init
   ```
3. Crea un archivo:
   ```bash
   echo "# Mi Proyecto" > README.md
   git add README.md
   git commit -m "Primer commit"
   ```
   
4. Verifica el historial:
   ```bash
   git status
   git log
   ```
